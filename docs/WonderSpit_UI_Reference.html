07 — Desktop · Proposta Tab FAQ
Terza voce nella sidebar — domande frequenti per guidare i nuovi utenti
☀️ Light
WonderSpit
Light
Come funziona?
🖊 Cosa scrivo nel prompt?
Descrivi quello che vuoi vedere: un animale, un paesaggio, una scena. Più dettagli dai, meglio è. Puoi anche usare i tag "Dettagli visivi" per arricchire automaticamente la ricerca.
⏱ Quanto tempo ci vuole?
Di solito 15–30 secondi. In momenti di alto traffico può richiedere qualche secondo in più — l'app riprova automaticamente.
🛒 Posso usare le immagini sui prodotti?
Sì! Dopo aver generato l'immagine, clicca "Usa nel Designer" per personalizzare un prodotto WonderSpit direttamente con la tua immagine AI.
🎨 Cos'è lo stile artistico?
Definisce l'aspetto visivo dell'immagine: Foto per risultati realistici, Digital per arte digitale, Anime per stile giapponese, e così via.
💾 Dove vanno le immagini generate?
Vengono salvate automaticamente nella Galleria, accessibile dal menu a sinistra. Puoi scaricarle o riusare i prompt in qualsiasi momento.
